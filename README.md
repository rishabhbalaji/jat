# JAT (Job Application Tracker)

A sleek, Kanban-style Job Application Tracker built with React, Vite, TypeScript, and Supabase! 

Features include:
- Real-time cloud sync powered by Supabase
- Drag-and-drop Kanban board for visualizing job applications
- Optimistic UI updates
- Authentication (view-only mode for guests)

### Vibe coding this using Google Antigravity 🚀

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Setup `.env.local` with your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
4. Run the development server: `npm run dev`
